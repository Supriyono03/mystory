Tentang

Hai semuanya, nama saya Supri yono dan saya sering di panggil Supri. Saya suka dengan pemograman web sudah lama sejak tahun 2015 dan waktu itu baru memasuki Laravel 5 dan saya belum bisa pemograman apapun.

Singkat cerita, saya mengenal pemograman web dari teman kelas SMK juga seorang alumni S1 Telkom. Beliaulah yang memperkenalkan Laravel dan PHP kepada saya, karena di waktu itu Laravel sangat populer hingga saat ini.

Hingga saya tamat SMK N 1 Purwokerto hingga masuk ke STMIK Widya Utama barulah saya memulai untuk belajar pemograman web baik dari belajar HTML, CSS, JavaScript, PHP, jQuery, Bootstrap, dan masih banyak lagi.

Pada kelas 1 di semester 1 saya membuat web dengan Wordpress yaitu Sekolah Program, banyak sekali tantangan dan ilmu yang saya dapatkan disana. Pada tahun 2018 saya memulai untuk belajar kerangka kerja Laravel dan membuat blog dengannya. Tepat pada tahun 2021 diwaktu kuliah semester 1 di STMIK Widya Utama, saya mendapatkan datantangan baru untuk belajar Vue.js agar memiliki web dengan Single Page Application hingga masa pandemi tiba saya mengalami masalah baru. Kalau membuat web dengan Single Page Application tidak rama dengan Search Engine Optimization.

Hingga bulan Juni 2021 saya menemukan solusi yaitu menggunakan Nuxt.js agar blog saya bisa menggunakan Server Side Rendered, maka dari itu Laravel sebagai REST API dan Nuxt.js sebagai Client. Tahun 2021 saya akan memasuki Sekolah Program untuk fokus menjadi pembelajaran pemograman secara online.
